# StudyEasily

StudyEasily est une plateforme d'échange entre étudiant, une peu comme moodle mais avec des fonctionnalités qui les différencient. 
Il permet à chaque étudiant de s'inscrire et d'avoir un compte qui lui est propre, de pouvoir se logger et programmer ses taches, 
consulter son emploi du temps, remplir ses notes. Un système de chat est mise en place pour des discussions privées ou en groupe. 
Cette plateforme a pour but de faciliter la vie de l'étudiant utilisateur en lui permettant de mieux s'organiser.
